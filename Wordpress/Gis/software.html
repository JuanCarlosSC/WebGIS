Software

  • Video

En Global Integral Services desarrollamos software de diferentes tipos: Software para pruebas eléctricas, para el manejo y proceso de tableros electrónicos, software de control, administración y flujo de información, simulaciones, entre otros. Realizamos software agradable e interactivo y fácil de utilizar; pero siempre enfocándonos a cubrir las necesidades del cliente, utilizando diferentes lenguajes de programación y siguiendo estándares de calidad.


El proceso de desarrollo está basado en lo siguiente:
1.- Definicíón y análisis de requerimientos: Ésta fase está encargada de establecer las diferentes necesidades del cliente, el cual realiza la descripción de su problema. En la cual se realiza la especificación de requerimientos.
2.- Arquitectura y diseño: En ésta etapa el proceso se enfoca a la implementación de patrones de diseño que se implantará en la aplicación. Además se establecen componentes a utilizar en el desarrollo del producto.
3.- Desarrollo: Dentro de ésta fase se realiza la codificación del software en base a diagramas y documentos acordados con el cliente en la fase de análisis.
4.- Prueba y validación: Se realizan las pruebas necesarias y se presenta ante el cliente para realizar pruebas de operación sobre el producto de software para verficación de la funcionalidad total. Seguido de ésto la instalación del sistema o la entrega de éste de acuerdo al trato acordado.
